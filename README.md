# chart-project
A playground to build charts in D3.

开始:

`npm i`

`npm start`



## 目录结构

```
.
├── index.html---------------------  html入口
└── src  --------------------------  源码目录
    ├── index.js  -----------------  入口js文件
    ├── test-name.js  -------------  逻辑代码
    ├── test-name.styl ------------  css文件
```
